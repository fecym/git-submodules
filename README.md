# git-submodules

## Clone project

```
git clone git@github.com:fecym/git-submodules.git --recursive
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
